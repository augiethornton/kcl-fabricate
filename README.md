KCL-Fabricate
=========

Kinesis Client Library (KCL) Data Producer for Test Automation

## Installation

  npm install kcl-fabricate --save

## Usage

  const kclFabricate = require('kcl-fabricate')

  kclFabricate.fabricate()

## Tests

  npm test

## Linting

  npm run eslint

## Contributing
